# Periscope_recons

This is a simple project to reproduce the result from *Periscope: A Keystroke Inference Attack Using Human Coupled Electromagnetic Emanations*:
```
@inproceedings{jin2021periscope,
  title={Periscope: A Keystroke Inference Attack Using Human Coupled Electromagnetic Emanations},
  author={Jin, Wenqiang and Murali, Srinivasan and Zhu, Huadi and Li, Ming},
  booktitle={Proceedings of the 2021 ACM SIGSAC Conference on Computer and Communications Security},
  pages={700--714},
  year={2021}
}
```
